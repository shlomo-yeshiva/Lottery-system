# 🎲 מערכת הגרלות מתקדמת לקמפיין פורים

<div dir="rtl">

מערכת הגרלות מתקדמת ומרשימה עם אפקטים ויזואליים מדהימים, בנויה במיוחד לקמפיין פורים. המערכת מאפשרת הגרלה הוגנת ומרשימה עם אפקטים מיוחדים, זיקוקים בסגנון Google Doodle, וניהול מלא של רשימת הזוכים.

## ✨ תכונות עיקריות

- 📊 **טעינת שמות מקובץ Excel** - העלה קובץ Excel והשמות יועלו אוטומטית מהעמודה הראשונה
- 🎨 **עיצוב מרשים** - ממשק משתמש מודרני עם אפקטים ויזואליים מתקדמים
- 🎆 **זיקוקים בסגנון Google Doodle** - אפקטים מרהיבים בעת חשיפת הזוכה
- ⏱️ **ספירה לאחור** - אפקט דרמטי לפני תחילת ההגרלה
- 🎯 **שמות רצים** - אנימציה של שמות רצים על המסך לפני בחירת הזוכה
- 🏆 **מעקב אחר זוכים** - רשימת כל הזוכים עם זמן ההגרלה המדויק
- 🎲 **הגרלת זוכים מרובים** - אפשרות להגריל מספר זוכים ברצף
- 🖼️ **רקע ולוגו מותאמים** - תמיכה ברקע ולוגו מותאמים אישית
- 📱 **עיצוב רספונסיבי** - מותאם לכל הגדלי מסך (דסקטופ, טאבלט, מובייל)

## 🚀 התקנה ושימוש

### דרישות מערכת

- דפדפן מודרני (Chrome, Firefox, Safari, Edge)
- חיבור לאינטרנט (לטעינת ספריות חיצוניות: XLSX.js, Google Fonts)

### התקנה

1. שכפל או הורד את הפרויקט:
   ```bash
   git clone https://github.com/shlomo-yeshiva/Lottery-system.git
   ```

2. פתח את הקובץ `index.html` בדפדפן

### שימוש בסיסי

1. **טעינת שמות**: לחץ על "📤 בחר קובץ Excel" והעלה קובץ Excel עם שמות
2. **התחלת הגרלה**: לחץ על "🎯 התחל הגרלה"
3. **צפייה באפקטים**: צפה בספירה לאחור, שמות רצים וזיקוקים מרהיבים
4. **הגרלת זוכים נוספים**: לחץ על "🎲 הגרל זוכה נוסף" כדי להגריל זוכים נוספים
5. **ניהול זוכים**: צפה ברשימת כל הזוכים עם זמן ההגרלה

### פורמט קובץ Excel

הקובץ צריך להיות בפורמט `.xlsx` או `.xls`, והשמות צריכים להיות בעמודה הראשונה של הגיליון הראשון.

**דוגמה:**
```
| שם        |
|-----------|
| דוד כהן   |
| משה לוי   |
| אברהם ישי |
```

## 🎨 התאמה אישית

### רקע מותאם

ניתן להעלות רקע מותאם אישית דרך הכפתור "🖼️ החלף רקע" בתפריט. הרקע יוצג עם שקיפות חלקית כדי לא להפריע לקריאות הטקסט.

### לוגו מותאם

ניתן להעלות לוגו מותאם אישית דרך הכפתור "🏛️ החלף לוגו" בתפריט. הלוגו יוצג בפינה הימנית העליונה עם אפקט ציפה עדין.

## 🛠️ טכנולוגיות

- **HTML5** - מבנה העמוד הסמנטי
- **CSS3** - עיצוב ואנימציות מתקדמות (Keyframes, Transitions, Gradients)
- **JavaScript (ES6+)** - לוגיקת ההגרלה ואפקטים דינמיים
- **XLSX.js** - קריאת קבצי Excel (מ-CDN)

## 🎯 דוגמאות שימוש

### הגרלה פשוטה

1. העלה קובץ Excel עם רשימת שמות
2. לחץ על "התחל הגרלה"
3. צפה בזוכה הנבחר

### הגרלת מספר זוכים

1. לאחר הגרלה ראשונה, לחץ על "הגרל זוכה נוסף"
2. חזור על הפעולה לפי מספר הזוכים הרצוי
3. כל הזוכים יישמרו ברשימה עם זמן ההגרלה

### איפוס והתחלה מחדש

- **איפוס**: לחץ על "🔄 איפוס" כדי להחזיר את כל השמות לרשימה
- **ניקוי זוכים**: לחץ על "🗑️ נקה זוכים" כדי למחוק את רשימת הזוכים

## 🐛 פתרון בעיות

### הרקע לא נטען
אם הרקע המובנה לא נטען, תראה הודעה. ניתן להעלות רקע מקומי דרך הכפתור "החלף רקע".

### קובץ Excel לא נטען
- ודא שהקובץ בפורמט `.xlsx` או `.xls`
- ודא שהשמות נמצאים בעמודה הראשונה
- ודא שהקובץ לא פגום

### אפקטים לא עובדים
- ודא שיש לך חיבור לאינטרנט (לטעינת ספריות)
- נסה לרענן את הדף
- בדוק את הקונסול לדיווחי שגיאות
